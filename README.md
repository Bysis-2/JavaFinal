## ✨ 遊戲特色
流暢的 2D 操作與動畫

支援滑鼠與鍵盤射擊

敵人會隨機生成並發射子彈

隨著分數增加，難度會自動提高

## 🕹️ 操作方式
滑鼠移動：控制飛船水平位置

滑鼠左鍵（長按）：連續射擊

左右方向鍵：鍵盤移動飛船

空白鍵（長按）：連續射擊

## 🔄 遊戲流程
主畫面：點擊「開始遊戲」開始

遊戲中：射擊敵人（得分 +1），躲避敵人子彈（血量 -1）

結束畫面：血量 ≤ 0 或敵人 > 25（失敗），得分 ≥ 450（勝利）

按鈕：再來一場、返回主畫面、離開遊戲

## 🔊 遊戲音效
背景音樂：遊戲開始時播放

射擊音效：玩家和敵人射擊時觸發

## 💀 結束條件
失敗：血量 ≤ 0 或敵人 > 25

勝利：得分 ≥ 450
