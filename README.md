**JavaFinal – 星際大戰 (StarWar)
*🎮 題目：StarWar
*👥 組員
*B11132040 蕭恩園
*B11032009 吳庭侑

✨ 遊戲特色
流暢的 2D 操作與動畫

支援滑鼠與鍵盤射擊

敵人會隨機生成並發射子彈

隨著分數增加，難度會自動提高

遊戲結束條件：

玩家血量歸零

場上敵人超過 25 隻

新增功能：

超過 100 分後，玩家可發射對角線子彈

每次難度提升時，玩家恢復 2 點血量

當分數達到 450 分時，遊戲勝利

🕹️ 操作方式
滑鼠移動：控制飛船水平位置

滑鼠左鍵（長按）：連續射擊

左右方向鍵：鍵盤移動飛船

空白鍵（長按）：連續射擊

敵人會立即發射一發子彈

碰撞檢查：

🛠️ 注意事項
對角線子彈的碰撞檢查需在 actionPerformed 中加入，確保其能正確擊中敵人並加分。

當分數達到 450 分時，應設定 gameOver = true 並顯示勝利畫面。

每次難度提升時，記得增加玩家血量 2 點，並確保不超過最大血量上限。
